## 2. 功能需求

### 2.1 用户管理模块

| 功能   | 描述           |
|------|--------------|
| 用户注册 | 用户填写基本信息注册账户 |
| 用户登录 | 支持用户名+密码登录   |
| 权限控制 | 区分普通用户与管理员权限 |

### 2.2 记账模块

| 功能   | 描述                       |
|------|--------------------------|
| 添加账单 | 记录收入/支出金额、类别、账户、时间、备注    |
| 修改账单 | 支持对已记录账单的修改              |
| 删除账单 | 支持删除指定账单记录               |
| 批量导入 | 支持从 Excel 或 CSV 文件导入账单数据 |

### 2.3 查询与统计模块

| 功能      | 描述                       |
|---------|--------------------------|
| 多条件查询   | 按时间、类别、账户、金额范围等条件筛选账单    |
| 月度/年度统计 | 自动生成收支统计报表               |
| 图表展示    | 支持柱状图、饼图等形式展示收支结构        |
| 导出报表    | 支持导出 Excel 或 PDF 格式的账单报表 |

### 2.4 分类与账户管理模块

| 功能     | 描述                |
|--------|-------------------|
| 收支分类管理 | 自定义收入与支出类别        |
| 账户管理   | 管理现金、银行卡、支付宝等账户信息 |

### 2.5 系统管理模块（管理员）

| 功能      | 描述              |
|---------|-----------------|
| 用户管理    | 查看、禁用、删除用户账户    |
| 数据备份与恢复 | 定期备份数据库，支持数据恢复  |
| 日志管理    | 查看系统操作日志，保障系统安全 |
